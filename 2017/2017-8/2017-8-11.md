# 知识扩展
## OWIN
    OWIN在.NET Web Servers与Web Application之间定义了一套标准接口，OWIN的目标是用于解耦Web Server和Web Application。IIS不足之处：        ASP.NET(System.Web)紧耦合IIS，IIS紧耦合OS，这就意味着，我们的Web Framework必须部署在微软的操作系统上，难以跨平台。OWIN可以解决这个    问题
## 单点登录系统（sso）
    解决问题：跨域共享登录，
    有时候一个网站中的页面是来自不同的域，在切换到不同域的页面的时候不应该要用户再重新登录一遍
