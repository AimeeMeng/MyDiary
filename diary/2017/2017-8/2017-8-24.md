# 知识扩展
## 网络热词
* 热加载
    服务中的文件（比如说类文件）变更时，无需重启服务，就能让变更生效
* Docker kubernetes 微服务
    微服务：把业务系统进行组件化和服务化，把原有的单个业务系统拆分为多个可以独立开发、
    设计、运行和维护的小应用程序，这些小应用程序之间通过服务完成交互和集成
    Docker：应用容器引擎
    Kubernetes：作为Docker生态圈中重要一员,是Google多年大规模容器管理技术的开源版本
* DevOps 自动化部署 Jenkins
    DevOps：（英文Development和Operations的组合）是一组过程、方法与系统的统称，用于促进
    开发（应用程序/软件工程）、技术运营和质量保障（QA）部门之间的沟通、协作与整合。它
    的出现是由于软件行业日益清晰地认识到：为了按时交付软件产品和服务，开发和运营工作必
    须紧密合作
    自动化部署：
    Jenkins：是一个开源软件项目，是基于Java开发的一种持续集成工具
